PLC Basic programming

Pre-request: General Knowledge about electricity and classic control

Course duration: 36 Hours

On completing this course you will be able to:

  • Understand all basics of any control panel and its complicated connections
  • Able to connect any production line with classical control
  • Understand the internal hardware of any PLC device
  • Able to connect any digital input or output
  • Learning basics of ladder programming
  • Able to make the program for any production line
  • Able to work on the software of Allen Bradley company
  • Able to work on the software of Siemens specially s7-1200
